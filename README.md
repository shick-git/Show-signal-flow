# Show Signal Flow — Concept Store
## Сборка .exe (Windows)

### Требования
- [Node.js](https://nodejs.org) v18+ (скачай и установи)
- Windows 10/11 x64

### Шаги

1. Распакуй этот архив в любую папку
2. Открой папку в командной строке (или PowerShell)
3. Выполни:

```
npm install
npm run build
```

4. Готовый `.exe` появится в папке `dist\`:
   - `ShowSignalFlow-v1.7-portable.exe` — портативный, запускается без установки
   - `ShowSignalFlow Setup 1.7.0.exe` — установщик с выбором папки

### Запуск в режиме разработки (без сборки)
```
npm start
```

### Форматы файлов проекта
- `.ssfp` — нативный формат (JSON), открывается только в приложении
- Экспорт в SVG, PNG, PDF — через кнопки в тулбаре
