    🔧 Авто-сеть — автоназначение IP
    ПринципАнализирует типы нод, определяет протокол, назначает IP в нужной подсети
    ArtNet / sACN→ 10.0.0.x / 255.0.0.0 (стандарт Art-Net 4 Spec)
    NDI / Видео→ 192.168.10.x /24 (выделенный VLAN для видео)
    Dante / Аудио→ 192.168.20.x /24 (отдельная сеть — требование Dante)
    OSC / Управление→ 192.168.1.x /24 (QLab, TouchDesigner, шоу-контроллеры)
    Интерком→ 192.168.50.x /24 (Riedel, Clear-Com, Pliant)
    Прочие→ 192.168.100.x /24
    Уже настроенныеПропускаются по умолчанию (кнопка «Переназначить всё» — сброс)
    После примененияВсе IP доступны для ручного исправления в таблице

    GrandMA2 — сеть и интеграция
    ETH0192.168.0.x/24 — MA-Net2, sACN (сессия, ноды, бэкап)
    ETH12.x.x.x/255.0.0.0 — ArtNet (жёсткое ограничение MA2)
    OSC⛔ MA2 НЕ поддерживает OSC нативно
    Telnet TCP 30000Управление командной строкой: "Login user pass" → "Go Exec 201"
    TD → MA2Art-Net DMX Out CHOP (2.x.x.x) → DMX Remote в MA2
    Loopback (onPC + TD)KM-Test Loopback Adapter → 2.0.0.1/255.0.0.0
    GrandMA3 — сеть и OSC
    MA-Net3UDP Multicast 30020 · multicast 236.4.1.x
    OSC (нативный)UDP порт 8000 · адрес /cmd · строка-команда как аргумент
    OSC пример/cmd,s,"Go+ Exec 402" — следующий шаг на экзекьюторе 402
    OSC настройкаIn & Out → OSC → включить Receive Command, Echo для отладки
    ⛔ Запрещено192.168.33.x — зарезервировано MA3, нельзя использовать!
    TD → MA3OSC Out CHOP → MA3 IP:8000 · MA3 OSC Out → TD IP:порт
    Резерв MA3MA3 Web Remote: HTTP 80 · WebSocket 8080 · SFTP 22
    TouchDesigner — сетевые CHOPs
    OSC In CHOPПорт задаётся вручную (нет дефолта). Типично 8001, 9000
    OSC Out CHOP→ MA3: IP консоли, порт 8000 · Local Address: укажи NIC явно!
    DMX Out CHOP (ArtNet)UDP 6454 · NIC должен быть 2.x.x.x при работе с MA2
    NDI TOPmDNS 5353 · Discovery 5959 · Video 5960+ · IGMP Snooping нужен
    Dante Virtual SoundcardОтдельный NIC для Dante, не смешивать с NDI
    Несколько NICВсегда указывай Local Address в каждом CHOP!
    Windows FirewallМолча блокирует UDP — создай inbound rule для портов TD
    Оптимизация NICОтключить Large Send Offload и Interrupt Moderation на шоу-NIC
    Конфликты — что нельзя смешивать
    ArtNet broadcast + NDIРазные VLAN или ArtNet → Unicast режим
    NDI через несколько VLANНужен NDI Discovery Server (TCP 5959)
    Dante + NDI без IGMPФлуд multicast → джиттер Dante. Включить IGMP Snooping!
    IPv6 на шоу-NICОтключить — создаёт лишний multicast и мешает mDNS
